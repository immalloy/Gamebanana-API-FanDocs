# GET /Util/Generic/Discussions

**Discussions**

This endpoint returns a list of the latest discussions for the specified Game ID.

## Query Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `_nPage` | Yes | Integer. Page to get. Allowed values: >1 Default value: 1 |
| `_idGameRow` | Yes | Required. Integer. Game ID. |
