# GET /Util/Generic/Tags

**Tags by text**

This endpoint returns a list of tags containing the search query in their names.

## Query Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `_sTag` | Yes | Required. Enter the search query. Case insensitive.  Allowed values: string of 2 characters or more. |
| `_idGameRow` | No | Narrow down the search to a specific Game's ID. |
